# Chatbot
A simple chatbot built using Python(Flask) and HTML, CSS, JavaScript that interacts with users using keyword-based logic. This project demonstrates how a basic chatbot works without using any external APIs or AI models. 

Features
- Real-time chat interaction
- Keyword based response system
- Works completely offline
- Simple and clear user interface
- Frontend and Backend integration

Tech Stack
- Frontend: HTML,CSS,JavaScript
- Backend: Python(Flask)

Note: This project is under developing phase. The backend will further be enhanced by integrating APIs to provide more dynamic responses and hence improving user experience.
